User-agent-utils

Utilities for processing user-agent strings. Can be used to handle http requests in real-time or to analyze log files.

Welcome to the user-agent-utils library

The user-agent-utils can be used to display operating system or browser information or to create an os/browser switch. 

UserAgent userAgent = UserAgent.parseUserAgentString(request.getHeader("User-Agent"));
